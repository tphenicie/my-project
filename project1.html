Tactical Comms Project #1

Mobile Communications Server

For this project I created a mobile communications server that fits securely inside of small form factor pelican case.
Users are able to connect to the server accross a software defined network solution called ZeroTier.
The server was built using a Raspberry Pi 4 with Ubuntu server and an application called TAK server.
TAK server provides a graphical interface that shows where each user is located on a map and provides a chat service.
In addition to the server, a GLI router was installed with a four port switch to allow for radio integration on the server.
The radio intergration will suport mesh networking radios that extend the use of the server to devices without internet connectivity.

Goals

My goal for this project was to develop a mobile server that allowed for multiple users to connect accross a software defined network.

Future Plans

I plan to expand this project by adding support for LoRa WAN devices that will expand a cost effective mesh network accross the public inernet.
